## `altered`





### Examples

```js
key.altered('F major') // => ['Bb']
```



## `key.chords`





### Examples

```js
key.chords('C major') // => ['C', 'Dm', 'Em', 'F', 'G7', 'Am', 'Bo'
key.chords('major', false) // => ['C', 'Dm', ...]
```



## `scale`





### Examples

```js
key.scale('C major') // => ['C', 'D', 'E', ...]
```
